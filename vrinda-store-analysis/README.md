## Overview

The **Vrinda Store Annual Report 2022** is an interactive Excel dashboard project designed to analyze sales performance, customer demographics, order status, and sales channels for an e-commerce retail business.

The dashboard was created using Microsoft Excel with Pivot Tables, Pivot Charts, and Slicers to generate actionable business insights.

---

## Project Objectives

- Analyze annual sales performance.
- Compare orders and revenue trends across months.
- Understand customer demographics.
- Identify top-performing states.
- Evaluate sales channels.
- Track order fulfillment status.

## Key Insights

### Sales vs Orders Trend
- Monthly sales and order volume are analyzed together.
- Peak sales observed during the first quarter.
- Gradual decline toward year-end.

### Gender Analysis
- Women contributed **64%** of total sales.
- Men contributed **36%** of total sales.

### Order Status
- Delivered Orders: **92%**
- Returned Orders: **3%**
- Cancelled Orders: **3%**
- Refunded Orders: **2%**

### Top Performing States

| State | Sales |
|---------|---------|
| Maharashtra | 2.99M |
| Karnataka | 2.65M |
| Uttar Pradesh | 2.10M |
| Telangana | 1.71M |
| Tamil Nadu | 1.68M |

### Customer Demographics
- Adult women form the largest customer segment.
- Senior customers contribute a smaller percentage of orders.

### Sales Channels

| Channel | Contribution |
|----------|-------------|
| Amazon | 35% |
| Myntra | 23% |
| Flipkart | 22% |
| Ajio | 6% |
| Meesho | 5% |
| Nalli | 5% |
| Others | 4% |

   - Channel
   - Category
